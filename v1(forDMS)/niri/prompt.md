## 最终设计目标

不要模仿 macOS。不要模仿 Android。而是打造一套真正属于 Linux 的现代桌面：
Apple 的空间设计 + Material You 的颜色体系 + Noctalia 自身的极简设计

整体风格：
70% Apple, 20% Material You, 10% Noctalia

关键词：
Minimal, Elegant, Floating, Depth, Glass, Material, Soft, Quiet, Modern, Consistent

整体气质：
- 克制、高级、柔和、安静
- 没有 Linux 味，不像 macOS 仿制品
- 更像 Google 与 Apple 联合设计的一套桌面

====================================================

## 核心原则（最高优先级）

整个系统必须拥有统一设计语言。任何组件都必须像由同一个设计团队设计。

禁止：
- 不同圆角、不同阴影、不同动画、不同字体、不同图标风格、不同间距、不同颜色体系

必须统一。

====================================================

## Layout

整个桌面增加留白。所有窗口保持呼吸感。

- 窗口之间：12~18px
- 屏幕边缘：16~24px
- Dock：距离底部约18px
- 整体布局：Floating，不要拥挤

====================================================

## Window

取消明显边框。尽可能使用柔和自然的 Shadow 代替 Border，不要浓重。
窗口圆角：20~24px，保持一致。

====================================================

## Blur

不要全局 Blur。

仅以下组件允许 Blur (25~40px)：
- Dock
- Top Bar
- Notification
- Launcher
- Context Menu
- Popover

普通应用窗口不要 Blur。

====================================================

## Transparency

适度透明。不要全透明，不要磨砂过度。
目标：高级。不是炫技。

====================================================

## Colors

禁止纯黑。禁止大量 #000000。
使用 Material You Dynamic Color，根据壁纸自动生成 Accent。

Surface 使用多层级保证层级清晰：
Background -> Surface -> Surface Container -> Surface Variant -> Popover

====================================================

## Typography (字体配置)

统一使用现代、干净的无衬线字体，并配置严格的 Fontconfig 回退机制。

**核心字体：**
- 英文/数字：**Inter** (推荐使用 Variable 版本以精确控制字重)
- 中文回退：**Noto Sans CJK SC** (保持科技感) 

**字重与排版规范：**
- 标题/强调文字：SemiBold (600) 或 Medium (500)
- 正文/常规 UI：Regular (400)

**优化字体渲染：**
- Antialiasing: `rgba` (Subpixel AA)
- Hinting: `slight`
- LCD Filter: `default`

确保字体边缘平滑清晰，无彩边，在 2K 屏幕上达到极致的阅读体验。

====================================================

## Icons

禁止使用 Papirus。禁止混用多个图标主题。
统一使用 Material Symbols。如果系统限制，可选择最接近 Material 的现代图标。

====================================================

## Noctalia Shell 内置组件 (UI 规范)

**⚠️ 核心注意**：Dock、Top Bar、Notifications、Launcher **均由 Noctalia Shell 原生提供**。请直接通过 Noctalia 的配置文件或样式（如 QML/CSS）进行统一定制，**绝不要**引入任何第三方组件（如 Waybar、Mako、Dunst、Rofi 等），以保证系统的极致纯净和统一性。

**1. Dock (Noctalia 内部组件)**
- 样式：Floating Dock（悬浮）
- 尺寸：高度约 58~64px，距离屏幕底部 18px
- 圆角：22px
- 质感：轻微透明 + 轻微 Blur (25~40px)
- 不要生硬模仿 macOS Dock，保持特有的现代感。

**2. Top Bar (Noctalia 内部组件)**
- 样式：保持纤细、安静。
- 尺寸：高度 28~32px
- 质感：轻微透明 + 轻微 Blur。

**3. Notifications (Noctalia 内部通知中心/气泡)**
- 样式：轻薄，不要厚重。
- 圆角：与全局窗口圆角保持高度一致。
- 质感：柔和阴影 + 轻微玻璃质感。

**4. Launcher (Noctalia 内部启动器)**
- 样式：保持极简，不要花哨。
- 质感：增加 Blur 和柔和 Shadow，并在交互或焦点处点缀 Material You 的 Accent Color。

====================================================

## Animation

动画必须统一。目标：Apple 风格。不是快。而是自然。

推荐：
- Window Open：220ms
- Window Close：180ms
- Workspace：260ms
- Ease Out Cubic

不要弹跳，不要夸张。

====================================================

## Wallpaper

推荐：
- 苹果官方风格、Google Material You 风格、抽象、渐变、自然摄影、低饱和

禁止：
- 赛博朋克、高对比动漫、RGB 灯污染

====================================================

## Spacing

统一间距系统：4, 8, 12, 16, 24, 32
所有组件遵循这一套 spacing。

====================================================

## Shadow

统一阴影。柔和、低透明、产生景深、不要厚重。

====================================================
